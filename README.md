# 🛒 Ecommerce‑Esun Spring Boot 後端 API

這是使用 **Spring Boot 3** + **Spring Data JPA** + **PostgreSQL** 打造的簡易電商後端 REST API。  
支援商品、訂單等基本資源管理，可作為學習、實作練習或專案基礎模板。

---

## 🚀 功能說明

目前後端包含以下主要功能：

- RESTful 介面管理商品（Products）
- RESTful 介面建立訂單（Orders）
- 使用 JPA 自動生成資料表
- 連接 PostgreSQL 資料庫

---

## 🧠 技術棧

這個專案使用以下技術：

| 技術 | 版本/用途 |
|------|------------|
| Java | 17 |
| Spring Boot | 3.x |
| Spring Data JPA | ORM 資料存取 |
| PostgreSQL | 資料庫 |
| Maven | 建置工具 |
| Hibernate | JPA 實作 |
| REST API | JSON 介面 |
